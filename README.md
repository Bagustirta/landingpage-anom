# Anom Elektronik Landing Page

Landing page modern dan responsif untuk Anom Elektronik, toko ritel produk elektronik rumah tangga, kelontong, dan gawai/gadget.

## Tech Stack

*   **Framework**: Vue 3 (Composition API / SFC)
*   **Build Tool**: Vite
*   **Styling**: Vanilla CSS (kustomisasi penuh)

## Fitur Halaman

*   **Desain Modern & Responsif**: Tampilan elegan dengan tata letak adaptif (mobile-friendly).
*   **Navigasi Interaktif**: Menu navigasi yang responsif terhadap scroll pengguna.
*   **Katalog Kategori**: Showcase kategori produk (Elektronik RT, Kelontong, Gadget) dengan visualisasi daftar produk yang dijual.
*   **Testimoni & Brand**: Slider/grid untuk menampilkan ulasan pelanggan dan merk-merk produk yang tersedia.
*   **Halaman Hubungi Kami**: Kontak detail, jam operasional, dan peta lokasi interaktif.

## Cara Menjalankan Project

### Prasyarat
Pastikan komputer Anda sudah terpasang:
*   [Node.js](https://nodejs.org/) (Versi 18 ke atas)

### Langkah Pengoperasian

1.  Masuk ke folder project:
    ```bash
    cd d:\laragon\www\landingpage
    ```
2.  Pasang dependensi:
    ```bash
    npm install
    ```
3.  Jalankan server development:
    ```bash
    npm run dev
    ```
4.  Buka browser Anda dan akses alamat lokal yang muncul (default: `http://localhost:5173`).
